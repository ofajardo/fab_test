About

SPA2MDH is a small web page to quickly upload SPA files to the MDH.

The files will be later uploaded to Teradata.

Login is required, contact the MDH for access.

For more information about the MDH please visit our DAC portal website.

This webpage was developed using the python package FlaskAppBuilder.

For questions or suggestions please contact the developers: Otto Fajardo and Iori Namekawa.
